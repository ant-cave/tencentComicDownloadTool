## 项目简介

这是一个基于Python的腾讯漫画下载工具，可以批量下载腾讯动漫平台的漫画章节。

## 功能特点

- 🔍 支持通过漫画ID或名称搜索
- 📚 批量下载多个章节
- ⚡ 多线程下载，提高下载速度
- 📊 实时进度显示
- 🔄 失败章节自动重试
- 🎨 彩色命令行界面

## 安装要求

### 系统要求
- Python 3.7+
- Windows/Linux/macOS

### 依赖安装

```bash
pip install -r requirements.txt
```

所需依赖：
- `requests` - HTTP请求库
- `quickjs` - JavaScript引擎
- `rich` - 终端美化库

## 使用方法

### 1. 运行程序

```bash
python main.py
```

### 2. 搜索漫画

程序启动后，您可以选择：
- **输入漫画ID**：直接输入漫画的数字ID
- **输入漫画名称**：输入漫画名称进行搜索

### 3. 选择章节

支持多种选择方式：
- 单个章节：`1`
- 多个章节：`1,2,4`
- 章节范围：`1-9`
- 混合选择：`1,2,5-10`

### 4. 下载

选择章节后，程序会自动开始下载：
- 使用4线程并发下载
- 显示实时进度条
- 自动重试失败章节

│   └── ...

## 技术实现

- **网络请求**：使用requests库处理HTTP请求
- **数据解析**：通过quickjs解析JavaScript代码获取漫画数据
- **并发下载**：使用ThreadPoolExecutor实现多线程下载
- **进度显示**：使用rich库提供美观的进度条
